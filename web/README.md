# Frontend (web)

Tecnologias sugeridas: Next.js + TypeScript + Tailwind CSS.

Scripts (a preencher ao criar o projeto):
- npm install
- npm run dev

Pontos importantes:
- Configurar PWA (manifest.json + service worker)
- Conectar à API (NEXT_PUBLIC_API_URL)
- Estrutura mínima: pages/login, pages/dashboard, pages/decks/[id]
